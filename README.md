# Yu_Launcher
WEB/Android/PC ゲームアシスタントソフトウェア

あいりすミスティリア等のDMM及びFanzaのゲームをDMM PLAYER経由で起動するとフレームレート60FPS固定や
Windows Subsystem for Linux環境で起動しないゲームも有るため
WebViewを使って60FPS以上 Windows機能依存無しを実現出来るDMM GAME PLAYERに代わるソフトウェア
